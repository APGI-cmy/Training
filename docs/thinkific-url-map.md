## How to Use This File

1. When a new unit is deployed, update the Vercel URL in the table above with the live address (e.g. `https://training-three-rosy.vercel.app/courses/vpshr-level-0/unit2/`).
2. Update the **Thinkific Lesson Title** column with the exact lesson name used in the Thinkific course builder.
3. Change the **Status** from `Coming soon` to `Live` once the lesson is published in Thinkific.
